CARZO – Used Car Marketplace System

Buying and selling used cars is a common need among people. However, there is no unified digital platform that efficiently connects used car dealers with potential buyers in a simple and transparent way. Customers often depend on offline visits, personal contacts, or unreliable listings to find suitable vehicles. This process consumes time and may lead to lack of trust, incomplete information, and poor decision-making.

This project proposes CARZO – a mobile-based Used Car Marketplace System, developed using Flutter and Firebase. The system provides a structured platform where verified car dealers can register their shops, upload car details with images, and manage listings, while users can browse available cars, search vehicles, view detailed specifications, and directly contact dealers.

The platform consists of three major modules: Admin App, Dealer App, and User App. The Admin module verifies dealer registrations and ensures authenticity. The Dealer module allows shop owners to manage their car inventory. The User module enables customers to explore used car listings and communicate with sellers seamlessly. This ensures transparency, reliability, and improved user experience.

Features:
  Admin Module:
  - Dealer verification system
  - Approve / Reject dealer registrations
  - View uploaded documents (ID Proof, GST Proof)
  - Manage dealer database
    
  Dealer Module:
  - Dealer registration and login
  - Upload shop details and verification documents
  - Add used car listings with images
  - Manage car inventory
    
  User Module:
  - Browse used car shops
  - View available cars dynamically
  - Search cars by name
  - View car full details with multiple images
  - Direct call option to contact dealer

Hardware Requirements:

Software Requirements

Conclusion
